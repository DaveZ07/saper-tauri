🚀 JAK URUCHOMIĆ:

DESKTOP (Windows):
- Zainstaluj MSI/NSIS z folderu bundle/
- Lub uruchom: npm run tauri:dev

MOBILE (PWA):
1. cd public && python -m http.server 8000
2. Na telefonie: http://192.168.1.140:8000
3. Chrome menu → "Dodaj do ekranu głównego"

ONLINE (GitHub Pages):
1. Stwórz repo na GitHub
2. Wrzuć pliki z public/
3. Włącz GitHub Pages
4. Zainstaluj jako PWA na telefonie

📱 PWA GOTOWE DO UŻYCIA:
• Serwer HTTP uruchomiony na porcie 8000
• Wszystkie pliki PWA w public/
• Responsywne na telefonach
• Działa offline
• Instalowalne jako natywna app
